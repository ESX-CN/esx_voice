# esx_voice
FXServer ESX Voice


#### 描述
这是一个近场语音控制脚本。默认快捷键是 `Left Shift + H` (QWERTY).

#### 下载

**1) 手动**
- 下载 https://github.com/ESX-CN/esx_voice/releases/latest
- 解压至 `resource/[esx]` 目录

**2) 使用 Git**

```
cd resouces
git clone https://github.com/ESX-CN/esx_voice
```

#### 安装

1) 添加 `start esx_voice` 至你的服务器配置文件 `server.cfg`
